一、W3C

二、MDN
MDN（Mozilla Developer Network ）是一个 wiki，即意味着任何人都可以向其添加和编辑内容。是一个汇集众多Mozilla基金会产品和网络技术开发文档的免费网站
https://developer.mozilla.org/zh-CN/

四、空元素
一个空元素（empty element）可能是 HTML，SVG，或者 MathML 里的一个不可能存在子节点（例如内嵌的元素或者元素内的文本）的element。
例子：<area>、<base>、<br>、<col>、<colgroup> when the span is present、<command>、<embed>、<hr>、<img>、<input>、<keygen>、<link>、<meta>、<param>、<source>、<track>、<wbr>

五、可替换元素
可替换元素（replaced element）的展现效果不是由 CSS 来控制的。这些元素是一种外部对象，它们外观的渲染，是独立于 CSS 的。简单来说，它们的内容不受当前文档的样式的影响。CSS 可以影响可替换元素的位置，但不会影响到可替换元素自身的内容。
例子：<iframe>、<video>、<embed>、<img>，在特殊情况下，下列元素可作为可替换元素处理：<option>、<audio>、<canvas>、<object>、<applet>
